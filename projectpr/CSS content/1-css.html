CSS is about enhancing the looks and feel or visual presentation of a webpage by adding various design techniques.

What is CSS?
CSS stands for Cascading style sheets. It means a stylesheet language that describes the presentation in terms of HTML or XML of a web page. CSS defines how elements should appear in terms of layout, colors, fonts, spacing, and other visual aspects.

There are two main ways to add CSS into your Webpage:

HTML Editor


This is called external CSS which seperates the CSS file to the HTML file

This is the Basic syntax of CSS:

Selector - These are the HTML elements or elements that you want to style.

Property - This is the attribute that you want to change (ex: color, font-size, background).

Value - The value you want to apply to the property (ex: red, 16px).
