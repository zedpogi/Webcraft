CSS Selectors

Selectors decides which HTML elements are affected by the CSS rules. Here are common types of selectors:

• Element Selector: Targets all elements of a specific type (e.g., p, h1, div).
• Class Selector: Targets elements with a specific class attribute (e.g., .class-name).
• ID Selector: Targets an element with a specific ID (e.g., #id-name).
