HTML Attributes

Let's say you want to show how to use a toy you're gifting someone. Instructions such as "Push the red button to make the toy move!" are similar to HTML attributes, which are additional characteristics about the items on a webpage.

HTML Attributes Example


In HTML, attributes provide the element more details and help control its behavior or appearance. Attributes are always specified in the opening tag of an element and come in name-value pairs.

Semantic HTML

A semantic element clearly describes its meaning to both the browser and the developer. Semantic HTML is like organizing your content into labeled boxes, so both people and computers can understand what each part is for.

Semantic HTML Example

<header> - This element represents introductory content or a group of navigational links.

<article> - This element is used to represent a self-contained piece of content that can stand alone, such as a blog post, news article, product description, or user comment.

<footer> - This element represents the footer of a section or the entire document. Footers generally appear at the bottom of a page or section.
